# 📌 PROMPTORIES (Prompt Management Station)

> **나만의 AI 프롬프트 템플릿과 실행 결과를 스마트하게 관리하는 라이브러리 서비스**

**PROMPTORIES**는 자주 사용하거나 성공적이었던 AI 프롬프트와 결과물(출력 예시)을 효율적으로 보관하고, 실시간 변수 조합 기능을 통해 복사하여 즉시 활용할 수 있는 최첨단 단일 페이지 웹 애플리케이션(SPA)입니다.

---

## 🚀 시작하는 방법

이 프로젝트는 별도의 데이터베이스나 백엔드 서버 설치 과정이 필요하지 않은 **순수 클라이언트 사이드 웹 애플리케이션**입니다. 

### 1. 즉시 실행하기
프로젝트 폴더에 있는 `index.html` 파일을 더블 클릭하여 웹 브라우저(Chrome, Edge, Safari 등)에서 즉시 실행할 수 있습니다.
*   **파일 위치**: `C:/Users/HRDPRO/Documents/antigravity/index.html`

## ✨ 핵심 기능 가이드

### 1. 실시간 검색 & 스마트 필터링
*   **초고속 검색**: 제목, 내용, 메모, 태그 전체를 아우르는 실시간 필터링을 지원합니다.
    *   **단축키**: 웹 페이지 어디서나 키보드 `/` 키 또는 `Ctrl + K` (Mac: `Cmd + K`)를 누르면 검색창으로 바로 포커스됩니다.
*   **카테고리 & 태그 브라우징**: 사이드바에서 카테고리별 등록 건수와 인기 태그를 확인하고 클릭 한 번으로 필터링할 수 있습니다.
*   **즐겨찾기 (Favorites)**: 자주 쓰는 프롬프트는 별(★)을 눌러 상단 즐겨찾기 탭에서 따로 모아볼 수 있습니다.

### 2. 스마트 템플릿 변수 엔진 (자동 감지)
*   프롬프트 본문에 `{{변수명}}` 형태(예: `{{주제}}`, `{{language}}`)로 텍스트를 입력하면, 상세 보기창에 **실시간 입력 폼이 자동으로 생성**됩니다.
*   입력 폼에 값을 채워 넣으면 실시간으로 조합된 **최종 완성본 프롬프트**를 실시간으로 확인하고 클릭 한 번으로 복사할 수 있습니다.

### 3. 데이터 백업 & 안전한 보관 (JSON 포터블)
*   **자동 저장**: 모든 데이터는 브라우저 내부의 안전한 저장 공간인 `localStorage`에 자동 보관되므로 브라우저를 닫아도 지워지지 않습니다.
*   **백업 (JSON 내보내기)**: 사이드바 하단의 **백업** 버튼을 누르면 전체 프롬프트 데이터를 `.json` 파일로 소장할 수 있습니다.
*   **복원 (JSON 가져오기)**: **복원** 버튼을 눌러 소장하고 있던 백업 파일을 업로드하면 데이터를 그대로 이어서 사용하거나 병합할 수 있습니다.
*   **초기화**: **초기화** 버튼을 통해 언제든 기본 한글 샘플 프롬프트 5종이 들어간 최초 상태로 되돌릴 수 있습니다.

## 🛠️ 기술 사양 (Tech Stack)

*   **Markup**: HTML5 (웹 접근성과 SEO 가이드라인을 준수하는 시맨틱 구조)
*   **Style**: Vanilla CSS3 (네온 그라데이션, Glassmorphism, 최적화된 화면 슬라이딩 애니메이션 적용)
*   **Script**: Vanilla JavaScript (ES6+ 최신 스펙, 정규식 기반 변수 치환 렌더러 탑재)
*   **Icons**: `Lucide Icons` (CDN 탑재)
*   **Fonts**: Google Fonts `Inter` 및 `Outfit`, 개발 코드를 위한 `D2Coding` 폰트 적용
